# Code.JS Pizza Application

### Node server setup

Install express and bodyParser (https://github.com/expressjs/body-parser)

$ npm install express

$ npm install body-parser


### Front-end

Front part is built using:

Bootstrap v3.3.5

jQuery v1.11.3

jQuery Easing v1.3 plugin

Ionicons

mustache JS


### Start

Run application:

$ npm start

Enjoy!
